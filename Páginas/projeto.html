Objetivo:

O site "TERROR" foi desenvolvido para mostrar o funcionamento e interação entre páginas web, trazendo os principais diretores do gênero e os filmes mais populares.

Público Alvo:

O site é voltado para quem curte filmes de terror.

Requisitos:

Para acesso ao conteúdo do site, é necessário um navegador web, como ( Mozilla Firefox, Internet Explorer ou Google Chrome ).

Tecnologias Utilizadas:

O site foi criado em HTML e CSS, utilizando o editor " Notepad++ ".

As fotos utilizadas foram capturadas da web e editadas.
